# 認識網路概念 FTP 是什麼？

FTP 就是 File Transfer Protocol (client and server)，它是一種獲得網際網路世界普遍採用的通訊協定之一。

文件傳輸協議（也稱為 FTP）已有 40 多年的歷史了！
在 1985 年的時候有一組非商業組織的學者們，提出了一種開放的協定，提供給想要製作檔案傳輸相關應用的電腦軟體設計者們參考，
讓大家可以依照這個標準，獨立製作出支持 FTP 協定的檔案傳輸軟體，確又可以確保互相能夠相容.

它可跨操作系統，軟件包互操作，並允許您存儲和檢索任何類型的內容或數據。

FTP 是允許您管理文件並將文件從一個系統傳輸到另一個系統的協議。例如從 FTP 客戶端到 FTP 服務器。

![](https://i.imgur.com/jcpUuRx.png)

通過 FTP，您可以使用以下命令：上傳，下載，複製，移動和重命名文件。

![FTP 範例 — 檔案文件](https://i.imgur.com/jKMAnY6.png)

使用 FTP 文件傳輸的缺點是，它沒有被加密。

數據安全性現在是一個熱門問題，可能會阻止某人使用 FTP。
但是，還有一些用於加密傳輸中文件的選項，包括 SFTP（安全文件傳輸協議或 SSH 文件傳輸協議）。
