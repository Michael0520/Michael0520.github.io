## template

從上一份筆記中，可以觀察到如何建立 template ，也就是使用樣板字面詞

```html

<alert></alert>

<script type = "module">
const app = Vue.createApp({
    data() {
      return {
      };
    },
  })
    .component('alert', {
    data() {
      return {
        text: '內部文字'
      };
    },
    template: `<div class="alert alert-primary" role="alert">
      {{ text }}
    </div>`
  });

</script>
```

## x-template

1. 首先在 `Vue` 物件實體後方新增一個 `app.component()` 的方法
2. 方法內前面是要輸入新建立的元件名稱，後面的物件要放的是第一個屬性是 `template:'#alert-template'`，如下方所示。
3. 在原本的 `<script>` 標籤上建立一個新的 `<script>` 標籤，type 屬性為 `text/x-template`，其 `id` 名稱對應新建立的元件 `template` 名稱，要使用 `#` 字號，因為對應的是 HTML 的 `id`。
4. 在 HTML 中加入該新元件的名稱標籤。

```html
<script type="text/x-template" id="alert-template">
    <div class="alert alert-primary" role="alert">
    x-template 所建立的元件
	  </div>
</script>

<alert2></alert2>

<script type="module">
  // 注意，這段起手式與先前不同

  const app = Vue.createApp({
    data() {
      return {
      };
    },
  })

  app.component('alert2',{
    template:`#alert-template`
  })

  app.mount('#app');
</script>
```

# 元件運用

## 直接帶入 template 標籤名稱即可

直接使用標籤帶入即可

```html
<h3>元件運用</h3>
<h4>直接使用 標籤 名稱</h4>

<alert></alert>
```

```jsx
const app = Vue.createApp({
    data() {
      return {
        componentName: 'alert'
      };
    },
  })

  app.component('alert', {
    template: `<div class="alert alert-primary" role="alert">
      範例ㄧ
    </div>`,
  });

  app.mount('#app');
```

## 搭配 v-for 指令

在資料中建立一個 array ，使用 v-for 指令來針對 array 資料狀態，做渲染幾次 template

```html
<h4>搭配 v-for 也是沒問題的</h4>
<alert v-for="(item, index) in array" :key="index"></alert>
```

```jsx
const app = Vue.createApp({
    data() {
      return {
        array: [1, 2, 3],
        componentName: 'alert'
      };
    },
  })

  app.component('alert', {
    template: `<div class="alert alert-primary" role="alert">
      v-for 範例
    </div>`,
  });

  app.mount('#app');
```

# v-is 綁定

## 單純使用 v-is

要注意的是 v-is 綁定的資料如果沒有加上額外的標記，會去指向變數，

而此範例的 template 被命名為 alert，應該在額外加上字串做標記 `v-is="'alert'"`

```html
<h4>使用 v-is 綁定</h4>
<div v-is="'alert'"></div>
```

```jsx
const app = Vue.createApp({
    data() {
      return {
        componentName: 'alert'
      };
    },
  })

  app.component('alert', {
    template: `<div class="alert alert-primary" role="alert">
      v-is 範例
    </div>`,
  });

  app.mount('#app');
```

## 動態屬性

- input 綁定資料 :
    - 因為加上 `v-model`，所以內容會去綁定資料集內的變數 componentName，而 componentName 內容為 'alert'，所以 input 內會有 alert 的字串
- div 綁定資料 :
    - 因為加上 `v-is` ，所以會去綁定上 componentName ，而 componentName 內容為 'alert'，所以等同於 `v-is="'alert'"`，會去渲染出 alert 名稱的 template

```html
<h4>動態屬性</h4>
<input type="text" v-model="componentName">

<p>任何標籤均可搭配 v-is 進行動態切換</p>
<div v-is="componentName"></div>
```

```jsx
const app = Vue.createApp({
    data() {
      return {
        componentName: 'alert'
      };
    },
  })

  app.component('alert', {
    template: `<div class="alert alert-primary" role="alert">
      v-is 動態屬性 範例
    </div>`,
  });

 app.component('alert2', {
    template: `<div class="alert alert-primary" role="alert">
      x-template 範例
    </div>`,
  });

  app.mount('#app');
```

## 動態標籤實戰技巧

由於 `<tbody>` 裡頭只能加上 `<table>`，
所以此時 table-row 的 template 名稱，就不能被使用，
為了開發便利，一定得加上 table-row 的 template 標籤，該怎麼辦呢？

可以使用 v-is 來做綁定標籤，就能夠正確被渲染了！

```html
<h2>動態標籤實戰技巧</h2>
<table>
  <thead>
    <tr>
      <th>標題</th>
      <th>內文</th>
    </tr>
  </thead>
  <tbody>
    <table-row></table-row>
    <tr v-is="'table-row'"></tr>
  </tbody>
</table>
```

```jsx
const app = Vue.createApp({
    data() {
      return {
      };
    },
  })

  app.component('table-row', {
    template: `<tr>
      <td>$</td>
      <td>這是一個 tr 項目</td>
    </tr>`
  });

  app.mount('#app');
```
