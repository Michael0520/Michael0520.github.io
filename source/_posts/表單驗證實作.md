## HTML
### 首先 html 先建立 container  來包裹表單
```htmlembedded=
<div class='container'>
<form>

</form>
</div>
```
### 建立表單標籤,加上 id 為了之後可以控制整個表單(繼承原理)，加上標題
**(命名很重要，避免找不到)**
```htmlembedded=
<div class='container'>
<form id='form' class='form'>
<h2>Register to Us</h2> // 標題

</form>
</div>
```
### 開始建立表格
```htmlembedded=
<div class='container'>
<form id='form' class='form'>
<h2>Register to Us</h2> // 標題
<!--  username -->
<div class="form-control">
          <label for="username">Username</label>
          <input type="text" 
          class="username"
          id="username"
          placeholder="Enter username" />
          <small>Error message</small> // 錯誤要顯示的名字
        </div>
<!--  email -->
<div class="form-control">
          <label for="email">Email</label>
          <input type="text" 
          class="email" 
          id="email"
          placeholder="Enter email" />
          <small>Error message</small>
        </div>
<!--  password -->
        <div class="form-control">
          <label for="Password">Password</label>
          <input
            type="password"
            class="Password"
            id="password"
            placeholder="Enter password"
          />
          <small>Error message</small>
        </div>
<!--  confirm password -->
        <div class="form-control">
          <label for="confirmPassword">Confirm email</label>
          <input
            type="password"
            class="confirmPassword"
            id="confirmPassword"
            placeholder="Enter password again"
          />
          <small>Error message</small>
        </div>
<!--   加入button      -->
    <button type='submit'>Submit</button>
</form>
</div>
```

## JS

### 先將會需要動到的標籤綁定起來

```javascript=
const form = document.getElementById('form')
const username =document.getElementById('username')
const email =document.getElementById('email')
const password =document.getElementById('password')
const confirmPassword =document.getElementById('confirmPassword')
```
### 設定監聽事件

```javascript=
form.addEventListener('submit',(e)=>{
  e.preventDefault(); // 讓此 submit 只執行一次
//   如果表格沒有填任何字串，就會呼叫 showError function
//   如果表格有填任何字串，就會呼叫 showSuccess function
  if(username.value===''){
    showError(username,'Username is required')
  }else{
    showSuccess(username);
    
})
```
### 寫 Error , Success function

```javascript=
// 如果失敗將會 更改 formControl 的 classname
// 更改為 'form-control error'
// 'form-control error' 的Css 設定為跑出紅色邊框
// 再將 small(警告紅字)綁上 id
// innerText 加上文字為，(message)參數,
// 上方 if 判斷式 message = 'Username is required'
// 所以紅字就會變成'Username is required'
const showError = ((input,message)=>{
  const formControl = input.parentElement;
  formControl.className = 'form-control error';
  const small = formControl.querySelector('small');
  small.innerText = message;
})

// 成功會變成綠邊框
const showSuccess = (input=>{
  const formControl = input.parentElement
  formControl.className='form-control success'
})
```
### 完成後再加上其他填空欄位的 if 判斷式

```javascript=
if(email.value===''){
    showError(email,'Email is required')
  }else{
    showSuccess(email);
  }
  if(password.value===''){
    showError(password,'Password is required')
  }else{
    showSuccess(password);
  }
  if(confirmPassword.value===''){
    showError(confirmPassword,'ConfirmPassword is required')
  }else{
    showSuccess(confirmPassword);
  }
```

### 撰寫 email 驗證

1. 因為 email 有較多形式，參考網路上現有的完美寫法，
2. Google search :  [js email regax](https://stackoverflow.com/questions/46155/how-to-validate-an-email-address-in-javascript)
3. 我們使用這段 
```javascript=
const re = /^(([^<>()[\]\\.,;:\s@"]+(\.[^<>()[\]\\.,;:\s@"]+)*)|(".+"))@((\[[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}\])|(([a-zA-Z\-0-9]+\.)+[a-zA-Z]{2,}))$/;
    return re.test(String(email).toLowerCase());
```
4. 貼上程式碼，並命名寫為 function
```javascript=
const isValidEmail = (email=>{
  const re = /^(([^<>()[\]\\.,;:\s@"]+(\.[^<>()[\]\\.,;:\s@"]+)*)|(".+"))@((\[[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}\])|(([a-zA-Z\-0-9]+\.)+[a-zA-Z]{2,}))$/;
    return re.test(String(email).toLowerCase());
})
```
5. 更改 email 的 if 判斷式為
```javascript=
if(email.value===''){
    showError(email,'Email is required')
//     加上這個 else if 去判斷
//     如果符合 isValidEmail function
//     就會跳Error 顯示字串更改為 is not valid
  }else if (!isValidEmail(email.value)) {
    showError(email,'Email is not valid')
  }
  else{
    showSuccess(email);
  }
```
