# 物件解構

以前我們要讀取物件裡面的值，都會如下程式碼來取值：

```javaScript
const myFamily = {
  father: 'Michael',
  mother: 'Eve',
};

// 一般會這樣寫
const father = family.father
const mother = family.mother
```

但藉由 ES6 的`物件解構`語法，下方程式碼也可達到一樣效果，讓程式碼可以更精簡：

```javaScript
const myFamily = {
  father: 'Michael',
  mother: 'Eve',
};

// 縮寫版
const { father, mother } = myFamily

// console.log(father)
// Michael
// console.log(mother)
// Eve
```

**沒有要用到全部屬性，也可以解構其中一個即可**，在 React 上，就很常利用這招，只取出自己要的功能語法來使用。

```javaScript
// 縮寫版
const { ming } = family
```

### 物件語法強化

以前我們要將變數代入物件屬性時，得用以下方式設計

```javaScript
const eggPrice = 30;
const eggNum = 9;
const total = 270;

const group = {
    eggPrice: eggPrice,
    eggNum: eggNum,
    total: total
}
```

現在 ES6 物件語法強化(Enhanced Object Literal)，可以透過精簡語法，將變數重組為物件，結果也會和上方一樣。

```javaScript
const eggPrice = 30;
const eggNum = 9;
const total = 270;

const group = {
    eggPrice,
    eggNum,
    total
}
console.log(group);
// {"eggPrice":30,"eggNum":9,"total":270}
```

過往物件屬性要代入函式，會像是下方語法：

```javaScript
const obj = {
    a:function(num){
        return num
    },
    b:function(){
        return "b"
    }
}
```

但藉由 ES6 強化語法，可以精簡寫成如下，就不需要再使用 `function` 關鍵字，程式碼會產生一樣的結果

```javaScript
const obj = {
    a(num){
        return num;
    },
    b(){
        return "b"
    }
}
```

## Demo

將下方的 data，透過 ES6 物件優化寫法，讓程式碼可讀性更高

### data

```javaScript
const students = 30000;
const schoolName = "歡迎來到 Michael's Blog";
const courseList = ["HTML", "JS", "React"];

const hexschool = {
  schoolName,
  students,
  courseList,
  sayHello() {
    return ` = '歡迎來到 Michael's Blog！`;
  },
  addStudent() {
    hexschool.students += 1;
  },
};
```

使用物件解構語法來優化 `callSchoolDetail` 函式

```javaScript
const hexschool = {
  students,
  schoolName,
  courseList,
  sayHello() {
    return ` = '歡迎來到 Michael's Blog！`;
  },
  addStudent() {
    hexschool.students += 1;
  },
};

function callSchoolDetail(school) {
  const { schoolName, students } = school;
  console.log(`名稱為 ${schoolName}`);
  console.log(`學生數共有 ${students}`);
}
callSchoolDetail(hexschool);
```

請透過物件強化語法，在填空處宣告一個 total 物件，使用前面的變數，賦予物件屬性與方法，讓後面的 `console.log` 可以成立。

```javaScript
const a = 1;
const b = 2;
const c = function () {
  console.log(`hello`);
};

const total = { a, b, c };

console.log("a: ", total.a);
// 1
console.log("b: ", total.b);
// 2
console.log("c: ", total.c);
// hello
```
